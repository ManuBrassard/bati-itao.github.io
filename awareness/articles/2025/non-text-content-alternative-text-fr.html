Protéger vos yeux – Outils pour votre confort visuel

Le texte Alt fournit une description détaillée des images pour les utilisateurs de technologies d’adaptation et est nécessaire chaque fois que vous utilisez quelque chose d’autre que du texte brut. Il s’agit d’une technique essentielle qui rend le contenu numérique accessible, car elle fournit des descriptions significatives des images.

Texte Alt – une exigence en matière d’accessibilité

Vous souhaitez maîtriser le texte Alt pour rendre votre contenu plus accessible dans vos applications Microsoft? Pour une démonstration rapide, consultez la vidéo Microsoft Support sur la manière d’améliorer l’accessibilité avec un texte Alt.
Voici quelques conseils :

Quand utiliser le texte Alt

  1. Pour les images informatives, fournissez un texte Alt concis pour décrire les renseignements clés.
  2. Pour les images textuelles, telles que les logos ou les bannières, le texte Alt doit inclure les mêmes mots ou le même texte que ceux qui apparaissent dans l’image.
  3. Les images qui n’améliorent que l’aspect visuel et n’ajoutent pas d’information significative à votre contenu doivent être marquées comme décoratives, de sorte qu’elles soient ignorées par les technologies d’adaptation.

Comment rédiger un texte Alt significatif

  1. Le texte doit être court et simple, en utilisant un maximum de 140 caractères en anglais et 170 en français. Utilisez un compteur de mots pour éviter de dépasser la limite.
  2. Ne décrivez la couleur que si elle aide le public à comprendre l’image.
  3. Ne commencez pas le texte par des mots tels que « image de » ou « photo de ».
  4. Évitez les hyperliens.

Comment rédiger un texte Alt pour des images complexes nécessitant une longue description

Pour les images complexes (infographies, graphiques, tableaux, cartes) et les longues descriptions, des étapes supplémentaires sont nécessaires pour garantir une accessibilité totale :

  • Dans le texte Alt : Soyez concis. Si des détails supplémentaires sont nécessaires, indiquez à quel endroit les utilisateurs peuvent trouver les descriptions longues (par exemple : « description longue disponible dans les notes d’allocution »).
  • Créer des descriptions longues :
    • Les descriptions longues doivent être aussi détaillées que nécessaire pour transmettre pleinement le sens de l’image.
    • Voici comment Inclure de longues descriptions pour les images complexes.

Comment utiliser un texte Alt dans un contexte bilingue

Le texte Alt bilingue n’est pas accessible. Le texte Alt doit être unilingue. Même si une image ne comporte pas de texte, le texte Alt utilisé doit être rédigé dans une seule langue. Lors de l’utilisation d’une image bilingue, il est important de dupliquer l’image et de créer un texte Alt distinct dans chaque langue afin de garantir l’accessibilité et la clarté pour tous les utilisateurs.

Améliorez l’accessibilité de votre contenu dès aujourd’hui. Utilisez un texte Alt pour chaque image afin de rendre votre contenu plus inclusif.
